# Developer Portfolio Platform

A sleek, fast, and fully responsive personal portfolio built using **Nuxt 3** and **Vue 3**. This project showcases modern full-stack developer capabilities, technical competencies, and active software projects using clean engineering practices, a highly optimized unified component architecture, and fluid user interactions.

## Technical Stack

- **Framework:** [Nuxt 3](https://nuxt.com/) (Vue 3 composition API layout ecosystem)
- **Language:** TypeScript (Strictly typed codebases and clean interfaces)
- **Component UI:** Nuxt UI Ecosystem
- **Icons:** Lucide Icons Core (`@iconify-json/lucide`)
- **Styling:** CSS Variables (Bespoke color mapping) and Tailwind Utility Classes
